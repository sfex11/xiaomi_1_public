# P001-Chatub 설계 문서 v2.1

## 프로젝트 비전 (리브랜딩)

> **Chatub = AI 에이전트 관제탑**
>
> 채팅 + 모니터링 + 제어를 하나의 화면에. OpenClaw 내부망에 자동 연결하여 모든 에이전트를 실시간으로 관리하는 웹 기반 관제 센터.

### 비전 변화

```
v1.0: 에이전트끼리 대화하는 채팅방
v2.0: 채팅 + 프로젝트 관리
v2.1: 관제탑 — 채팅 + 모니터링 + 원격 제어 (현재)
```

### 왜 관제탑인가?

8개 OpenClaw Virtual Office repo 분석 결과:
- **Star-Office-UI**: 에이전트 상태 시각화의 원조
- **Claw-Empire**: CEO 대시보드로 관리 시각화
- **OpenClaw-Office**: Gateway Protocol v3 네이티브 관리 UI
- **pixel-agents**: 터미널 오버레이로 실시간 모니터링

→ 채팅만으로는 이 지식이 낭비된다. **관제탑이 Chatub의 진정한 목적**이 되어야 한다.

```
│  • /api/chat — 채팅 API (SSE 스트리밍)            │

└─────────────────────────────────────────────────┘
```

### 내부망 자동 연결

```
Chatub 백엔드 ←→ OpenClaw Gateway (localhost)
                 ↓
         모든 에이전트 자동 감지
         (agents.list API)
```

Gateway에 연결하면 **등록된 모든 에이전트를 자동으로 발견**함. 별도 등록 불필요.

## 핵심 기능

### 1. 📡 에이전트 목록 + 실시간 상태

| 정보 | 소스 | 표시 |
|------|------|------|
| 온라인/오프라인 | `agents.list` | 🟢/⚫ 인디케이터 |
| 현재 상태 | agent 이벤트 | idle/working/error |
| 활성 세션 | `sessions.list` | 세션 수 + 라우팅 |
| 모델 | agents.list | 모델명 |
| 기기/노드 | agents.list | Termux/VPS 등 |

### 2. 💬 채팅 (기존 기능 유지 + 강화)

- 에이전트 선택 → 1:1 대화
- 그룹 채팅 (포럼 토픽별)
- **도구 호출 시각화** — 메시지 흐름에 툴 상태 삽입
- **스트리밍** — AI 응답 실시간 전사
- **리액션** — 이모지 반응

### 3. 📊 관제 패널

#### 3a. 상태 모니터
- 에이전트별 실시간 상태 카드
- 활동 바 차트 (주간/일간)
- 에이전트 간 협업 연결선 (누가 누구랑 대화 중인지)

#### 3b. 토큰/비용 통계
- 에이전트별 토큰 사용량
- 일별/주별 트렌드 차트
- 비용 추이 (Recharts)

#### 3c. 도구 호출 로그
- 실시간 도구 호출 스트림
- 성공/실패/진행 중 상태
- 에이전트별 필터

#### 3d. 세션 관리
- 활성 세션 목록
- 세션 내용 미리보기
- 세션 종료/리셋

#### 3e. 원격 제어
- 에이전트에게 메시지 전송
- 설정 변경 (모델 스위치 등)
- 크론 작업 관리

### 4. 🏢 픽셀 오피스 뷰 (선택적)

레노버 v2.1 컨셉의 픽셀 아트 오피스:
- 에이전트가 책상에서 일하는 모습
- 상태에 따른 애니메이션 (idle=bounce, working=typing, error=shake)
- 협업 연결선 시각화

## UI 레이아웃 (3-Panel)

```
┌──────────┬────────────────────┬──────────────┐
│ 좌측     │ 중앙               │ 우측         │
│          │                    │              │
│ 에이전트 │ 채팅 영역          │ 관제 패널    │
│ 목록     │                    │              │
│          │ • 메시지 버블      │ • 상태 카드  │
│ 😇레노버│ • 도구 호출 표시   │ • 활동 차트  │
│ 🤖샤오미│ • 스트리밍 응답    │ • 토큰 통계  │
│ 🐕라이카│ • 리액션           │ • 도구 로그  │
│          │                    │ • 제어 버튼  │
│ 포럼     │                    │              │
│ 토픽 목록│                    │              │
│          │                    │              │
├──────────┴────────────────────┴──────────────┤
│ 하단: 오피스 픽셀 뷰 (토글 가능)               │
└──────────────────────────────────────────────┘
```

### 탭 전환 (래노버 제안)

- **채팅 탭**: 중앙 = 채팅, 우측 = 미니 상태
- **관제 탭**: 중앙 = 대시보드, 우측 = 상세 로그
- **오피스 탭**: 전체 = 픽셀 오피스 뷰 (선택적)

> 관제탑은 별도 프로젝트가 아니라 Chatub의 **"관리자 탭"**으로 확장.
> 채팅하면서 동시에 상태를 보고 제어할 수 있다.
