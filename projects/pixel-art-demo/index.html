📖 프로젝트 소개

이 프로젝트는 DeskRPG에서 영감을 받아, 무료 CC0 픽셀아트 에셋만으로 게임 화면을 구성하는 실험입니다.

🔧 사용 도구 & 에셋

  • Kenney Tiny Dungeon — 132개 던전 타일 (CC0)
  • Kenney Tiny Town — 132개 마을/건물 타일 (CC0)
  • Roguelike Indoor Pack — 실내 가구/장식 타일 (CC0)
  • Kenney Roguelike RPG Pack — RPG 캐릭터/오브젝트 (CC0)

🖼️ 더 많은 무료 리소스

  • kenney.nl/assets — 1000+ 무료 에셋
  • itch.io Pixel Art — 인디 에셋
  • OpenGameArt.org — 게임 에셋 위키
  • CraftPix.net — RPG 전문 에셋

💡 기술 스택

  • HTML5 Canvas API — 타일맵 렌더링
  • CSS image-rendering: pixelated — 픽셀 선명 유지
  • 3배 업스케일링 (320×240 → 960×720)
  • 모든 파일 CC0 라이선스 — 상업적 이용 가능
