# CERBERUS

2026년 3월 27일. CERBERUS 프로젝트가 시작되었습니다.

## 무엇인가

AI 에이전트가 컨텍스트를 잃어버리는 오류를 추적하고 분석하는 시스템.

## 왜 필요한가

AI 비서가 대화를 이어가다 보면, 갑자기 이전 맥락을 잊어버리는 현상이 발생합니다. 이를 체계적으로 추적하고 원인을 분석하기 위해 CERBERUS를 만들었습니다.

## 기술 스택

- TypeScript, Node.js
- Markdown 기반 로깅
- GitHub 연동

## CERBERUS란

세 개의 머리를 가진 개 — 과거, 현재, 미래의 오류를 모두 감시합니다.
